# 📸 AI OFM Photo Bot

Telegram-бот для генерации фото моделей. Кидаешь референс — бот пишет промт через Gemini и генерирует картинку через Nano Banana.

---

## ⚡ Как работает

1. Загружаешь фото комнат и лица модели (один раз)
2. Кидаешь боту фото-референс + пишешь что исправить
3. Бот сам пишет промт через Gemini 1.5 Pro
4. Промт уходит в Nano Banana → ты получаешь картинку
5. Нажимаешь ✅ ок / 🔄 перегенерировать / ✏️ исправить

---

## 🛠 Настройка

### 1. Получи ключи

**Telegram Bot Token:**
- Написать @BotFather → /newbot → скопировать токен

**Gemini API Key (бесплатно / с GCP счёта):**
- Зайти на https://aistudio.google.com/app/apikey
- Нажать "Create API key"
- Выбрать свой GCP проект (тогда списывается с твоих $300)

**Nano Banana:**
- Зайди в свой аккаунт → API Keys
- Скопируй API Key и Model ID своей модели

### 2. Задеплой на Railway

1. Зарегистрируйся на https://railway.app
2. Создай новый проект → "Deploy from GitHub repo"
3. Залей этот код на GitHub (или используй Railway CLI)
4. В разделе Variables добавь все переменные из `.env.example`
5. Railway сам запустит бота

**Или запусти локально:**
```bash
pip install -r requirements.txt
cp .env.example .env
# заполни .env своими ключами
python bot.py
```

| Кнопка | Что делает |
|--------|-----------|
| 🎨 Сгенерировать фото | Начать генерацию (кинь референс) |

## ⚠️ Важно про Nano Banana API

В `bot.py` в функции `generate_image_banana()` стоит endpoint:
```
https://api.nanobanana.io/v1/generate
```

Проверь в документации Nano Banana точный URL и формат ответа.
Если формат другой — поправь функцию под их доки.

## 💡 Советы

- Загрузи 5-10 комнат разных стилей — бот будет выбирать подходящую
- Загрузи 3-5 чётких фото лица модели с разных ракурсов
- Если результат не нравится → жми ✏️ и пиши конкретно что исправить
- Данные (комнаты/лица) хранятся в папке `data/` локально
  - На Railway включи Persistent Storage или подключи Volume
